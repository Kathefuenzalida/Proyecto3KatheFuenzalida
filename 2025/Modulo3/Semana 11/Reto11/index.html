¿Qué son las Media Queries?

Las media queries son una característica de CSS que permite aplicar estilos diferentes según el tamaño de la pantalla o el dispositivo del usuario. Gracias a ellas, una página web puede adaptarse de forma responsiva a celulares, tablets, laptops o pantallas grandes, ofreciendo una mejor experiencia visual y de uso.
