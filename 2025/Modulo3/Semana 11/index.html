.container { display: grid; grid-template-columns: repeat(2, 1fr); grid-template-rows: repeat(2, 1fr); grid-gap: 10px; } .item { background-color: #ccc; padding: 20px; } En este ejemplo, la clase "container" define un contenedor que utilizará CSS Grid para crear una cuadrícula de dos columnas y dos filas. La propiedad "grid-template-columns" define el ancho de cada columna y la propiedad "grid-template-rows" define el alto de cada fila. En este caso, estamos usando la función "repeat()" para especificar que queremos dos columnas y dos filas con un ancho y alto igual. La propiedad "grid-gap" establece el espacio entre los elementos de la cuadrícula. La clase "item" define el estilo de los elementos individuales en la cuadrícula, en este caso, un fondo gris claro con un relleno de 20 píxeles. Puedes experimentar con diferentes valores de ancho, alto y espacio entre elementos para crear una variedad de diseños utilizando CSS Grid.
